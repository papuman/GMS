# Registro Tilopay Developer
Regístrese cómo desarrollador en Tilopay para recibir credenciales de sandbox y realizar pruebas con todas nuestras herramientas e integraciones. Adicionalmente, puede solicitar acceso a cuentas de comercios reales para configurar la cuenta de Tilopay de sus clientes y verificar sus transacciones.
