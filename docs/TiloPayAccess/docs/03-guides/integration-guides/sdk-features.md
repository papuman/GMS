# Tilopay SDK
Genere una experiencia personalizada de cobros con nuestra SDK para comercios

Esta integración proporciona la funcionalidad de usar en su formulario de pago la seguridad que proporciona Tilopay. El proceso de configuración de esta integración requiere de un conocimiento básico o intermedio en tecnologías Javascript y HTML.
