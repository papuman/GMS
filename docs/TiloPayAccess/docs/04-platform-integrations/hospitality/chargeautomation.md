# Tilopay <> ChargeAutomation
We are happy to announce that ChargeAutomation is integrated with Tilopay!


### Setting up your ChargeAutomation with Tilopay
- Login to your Tilopay account
- On the left panel, click on Tilopay Checkout
- Note down the details listed on that page: Api Key, Api User and Api Password
- ‼️IMPORTANT NOTE: To successfully process transactions please ensure you follow the steps here Also ensure you have Recurring Payments checked like below
- Login to your ChargeAutomation account here chargeautomation.com
- Go to Settings >> Gateway Connection and select Tilopay from the dropdown
- Fill out the details you copied above and click on "Continue"

República Dominicana

El Salvador

St. Maarten

Islas Vírgenes Britanicas

Trinidad y Tobago

Islas Caiman
