### 5. Troubleshooting Guides (`troubleshooting/`)

**Purpose:** Common issues and solutions
**Status:** ðŸ”„ Directory created, guides to be populated

**Common Issues:**

#### Issue: Invoice Rejected by Hacienda
**Symptoms:** Invoice status shows "rejected"
**Causes:**
- Invalid XML structure
- Missing required fields
- Invalid CABYS code
- Certificate expired

**Solutions:**
1. Check Hacienda response message for specific error
2. Fix the identified issue
3. Resubmit invoice
4. If certificate expired, upload new certificate

#### Issue: Certificate Expiration Warning
**Symptoms:** System shows certificate expires in < 30 days
**Solution:**
1. Contact BCCR to renew certificate
2. Download new .p12 certificate
3. Upload to Odoo (Settings â†’ E-Invoice â†’ Certificates)
4. Test with sandbox environment first

#### Issue: Email Not Sending
**Symptoms:** PDF generated but customer not receiving email
**Causes:**
- SMTP not configured
- Email template missing
- Invalid customer email

**Solutions:**
1. Check SMTP settings (Settings â†’ Technical â†’ Outgoing Mail Servers)
2. Verify email template exists
3. Check customer email address is valid

#### Issue: POS Offline Queue Not Syncing
**Symptoms:** Sales made offline not appearing in Hacienda
**Solutions:**
1. Check internet connectivity
2. Manually trigger sync (POS â†’ Offline Queue â†’ Sync)
3. Check for errors in offline queue records
