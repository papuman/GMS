## Troubleshooting (30-Second Fixes)

### Problem: "No memberships configured"

**Fix**: Reload POS
1. Click browser refresh
2. Or close and reopen POS tab

### Problem: Member not found

**Possible Reasons**:
- Misspelled name
- Member not in system yet
- Wrong email/ID

**Fix**:
- Try different search terms
- Create new member if needed

### Problem: Button not responding

**Fix**:
1. Wait 1 second
2. Try clicking again
3. If still stuck, reload POS (F5)

### Problem: Queue not syncing

**Fix**:
1. Check internet connection
2. Wait 1 minute (auto-sync every 30s)
3. If still stuck, call manager
